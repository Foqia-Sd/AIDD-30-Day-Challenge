# Quickstart Guide: Python CLI Calculator

This guide will help you get the Python CLI Calculator up and running.

## Prerequisites

- Python 3.x installed on your system.

## Running the Calculator

1.  **Clone the repository** (if you haven't already):

    ```bash
    git clone <repository-url>
    cd calculator
    ```

2.  **Navigate to the project directory**:

    ```bash
    cd path/to/your/calculator/project
    ```

3.  **Run the application**:

    ```bash
    python main.py
    ```

4.  **Follow the prompts** in the terminal to enter numbers and select an operation.

## Example Usage

```
Welcome to the Simple CLI Calculator!
Enter the first number: 10
Enter the operator (+, -, *, /): /
Enter the second number: 2
Result: 5.0
```
